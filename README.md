# bbsetup
                                                            
 Bug Bounty Hunting Tools is a script to install the most popular tools used while looking for vulnerabilities for a bug bounty program.      
       

# Tools

    dirsearch
    arjun
    paramspider
    x8
    nuclei
    ffuf
    dirbuster
    gobuster
    assetfinder
    subfinder
    github-subdomains
    reconftw
    GF-Patterns
    Waybackurls
    Httprobe
    Seclists collection


                                                          

# Installing

    git clone https://github.com/A1C3VENOM/bbsetup/install.sh
    cd bbht
    chmod +x install.sh
    ./install.sh
